# OSI参考模型

| 模型层     | 功能                                           |
| ---------- | ---------------------------------------------- |
| 应用层     | 针对特定的应用层协议                           |
| 表示层     | 将数据转化成网络通用格式                       |
| 会话层     | 建立和断开逻辑连接                             |
| 传输层     | 负责数据可靠传输                               |
| 网络层     | 寻址和路由，负责将数据发送至最终目标地址       |
| 数据链路层 | 负责发送一个分段内的数据（互连设备之间的传送） |
| 物理层     | 将比特流转化为电压和脉冲光传输给传输介质。     |
